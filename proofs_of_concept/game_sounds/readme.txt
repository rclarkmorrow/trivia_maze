This proof of concept requires the following dependencies
to be installed:

I highly recommend installing dependencies in a virtual
Python environment to avoid cluttering your system Python.

1. playsound

Install command:
    python3 -m pip install playsound

If you are using MacOS you may also need these modules to run
playsound:

1. PyObjC
2. PyObjC-core

Install command:
    python3 -m pip install PyObjC PyObjC-core
